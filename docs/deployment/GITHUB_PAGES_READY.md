## 🔧 How It Works

**Hybrid System:**
- Automatically detects if PHP backend is available
- Uses PHP/MySQL when available (local development)
- Falls back to localStorage when not (GitHub Pages)
- **No configuration needed!**

**Data Storage:**
- All data stored in browser localStorage
- Persists across sessions
- Pre-loaded with 24 products
- Admin account ready to use

## Installation Instructions

### For Users:

1. **Unpack the mouse**
2. **Insert battery** (1x AA, usually included)
3. **Remove USB receiver** from mouse storage slot
4. **Plug receiver** into USB port
5. **Turn on mouse** (switch on bottom)
6. **Wait for connection** (automatic, 2-3 seconds)
7. **Start using!**

No drivers needed - plug and play!

---

## Troubleshooting

### Mouse Not Working?
1. Check battery is inserted correctly
2. Ensure mouse is turned ON
3. Try different USB port
4. Check receiver is fully inserted
5. Replace battery if old

### Connection Issues?
1. Move receiver closer to mouse
2. Remove metal objects between mouse and receiver
3. Try different USB port
4. Check for interference from other wireless devices

### Cursor Jumping?
1. Clean mouse sensor (bottom)
2. Use on proper surface (not glass/mirror)
3. Adjust DPI settings
4. Check battery level

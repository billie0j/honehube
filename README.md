# Honehube — Evlyne Hone College Electronics Marketplace

Pure frontend — HTML, CSS, JavaScript. No backend or server required.
All data is stored in the browser's localStorage.

## How to run
Just open `frontend/index.html` in your browser. That's it.

Or with XAMPP: place the folder in htdocs and visit:
http://localhost/Honehub/frontend/index.html
